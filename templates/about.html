CLOUD COMPUTING

Di materi pembuatan web profile ini, Kita diarahkan untuk mengeksekusi HTML dan CSS sebagai Front-end Web, dan penyambungan route menggunakan python agar dapat menjalankan web secara dinamis. Metode penyambungan route ini berupa Flask yang penggunaannya menggunakan python. Selain itu, kita harus menyambungkan web profile kita dengan database Microsoft Azure. Untuk rule yang diperlukan seperti dibawah:

1. Pembuatan web profile dengan HTML dan CSS

2. Penyambungan dan pengeditan code HTML untuk terhubung ke flask

3. Pembuatan file app.py untuk code route html

4. Penyambungan database ke Platform Microsoft Azure
